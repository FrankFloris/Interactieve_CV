Hier is een video van mijn nieuwste gameproject: Een 3D wereld

Voor deze 3D wereld heb ik gebruik gemaakt van de Lightweight Java Game Library (lwjgl). Door het gebruik van verschillende vectoren, matrices en shaders kan er een 3D wereld worden gerenderd op het beelscherm. Deze wereld heeft terrein met hoogte verschil, een skybox, een dag en nacht cyclus, meerdere lichtpunten en water. Het water reflecteerd afhankelijk van de camerapositie, recht van boven is er nouwelijks reflectie, maar van de zijkant is er zeer veel reflectie. Daarnaast is het mogelijk om entiteiten zoals bomen en bloemen op te pakken en ergens anders weer te herplaatsen. Uiteraard kunnen deze entiteiten ook vanuit het niets worden geplaatst. De laatste toevoeging die ik heb gedaan zijn vrolijk rondhuppelende konijntjes. Deze konijntjes zorgen momenteel nog voor wat performance issues, maar dat is gewoon een nieuwe uitdaging!

Java Chess

Dit is een schaakspel dat gebruik maakt van Java's JPanel. Het was een hele opgave om de regels van schaken te vertalen naar een spel dat je daadwerkelijk kan spelen op en computer. Momenteel kan je via hotseat met een tweede persoon spelen. Maar als je geen tegenstander hebt, dan kan je ook tegen de AI spelen. Deze AI maakt gebruik van het minimax algoritme. Het is ook mogelijk om de AI tegen een andere AI te laten spelen. De AI kan je in principe zoveel zetten vooruit laten denken als je wil, maar iedere extra zet die vooruit wordt gedacht vereist een extra factor aan berekeningen. De rekentijd kan zo erg lang duren, net zoals bij een gewoon potje schaak. Momenteel is het nog niet mogelijk om het spel op te slaan, of opgeslagen spellen te laden, maar dat kan nog worden toegevoegd.

Project Bokemon

Project Bokemon is een project dat ik samen met een medetrainee heb gemaakt. Bokemon is een Angular project dat gebruik maakt van frontend en een backend met een MySQL server. Het is feitelijk een Pokemon-kloon. Je kan met jouw personage rondlopen in een wereld die bewoond wordt door "Bokemon". Deze Bokemon kan je tegenkomen in het hoge gras, om er vervolgens mee te vechten. Als je ze weet te verslaan levert dat ervaringspunten op voor jouw Bokemon. Je kan deze Bokemon ook verzamelen door ze te vangen.

Wizard Shooter

Dit is een simpel spelletje. Je bent een tovenaar die door een dungeon loopt. Hier kom je vijanden tegen die jou schade doen als ze je weten te raken. Om dat te voorkomen kan je de vijanden uitschakelen door ze te bombarderen met jouw magische krachten. Maar let op, je bezit maar beperkte energie, dus schiet niet willekeurig in het rond. Gelukkig kan je jouw magie bijvullen met speciale kistjes.
